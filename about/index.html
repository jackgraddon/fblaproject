What Are AI Generated Images?

AI Generated images use an AI to produce images based off of a given prompt. At Generated, we use an AI named VQGAN, a generative adversarial neural network that is good at generating images that look like other images. But that doesn't get us images based off of a prompt, which is where CLIP comes in. CLIP is a neural network that is able to determine how well a string of words matches a given image. Together, these two AIs can generate images that offer familiar glimpses of reality, yet feel 'broken' somehow.

What is VQGAN?

VQGAN is a generative adversarial neural network, a type of machine learning where two neural networks compete with each other in a sort of game where one network's win is the other network's loss. In VQGAN, the game they play is "Who can generate the best image?" The winner of each round in the game is determined by CLIP (see below). A game's number of rounds is defined by the user, with the more rounds played meaning the higher quality the resulting image. Each network uses input on the previous round to build upon its generation skills, which gives us a more detailed image with more clarity provided as to what the image is.
